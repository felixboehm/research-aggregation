# Expected Combinations from the 3 Test Fixtures

## Must Be Recognized

### Combination: Location Strategy + Menu Optimization
- **Synergy:** Location-driven pricing (high-rent → premium menu required)
- **Synergy:** Format follows location (transit → grab-and-go, suburban → full menu)
- **Conflict:** Menu variety vs. service speed at high-traffic locations
- **Worked example:** From Doc C (rent ratio improvement from 48.5% to 27.8%)

## Should Be Recognized

### Combination: Pricing + Seasonal Strategy
- **Anchor pricing** + seasonal specials = higher perceived value
- **Seasonal items** have 15–25% higher margins
- Risk: Seasonal changeover costs

### Conflict: Volume vs. Variety
- **High-traffic locations** need fast service (under 3 min per order)
- **Large menus** slow down service
- Decision Tree: "High foot traffic? → Limited core menu + daily specials"
